# PETV Workflow (Plan, Explore, Task, Verify)

Adopt the multi-phase execution strategy:

## 1. Plan
- Define the end-state.
- Create atomic checklist.
- Identify risks.

## 2. Explore
- Search codebase for relevant patterns.
- Verify assumptions.
- Research external dependencies.

## 3. Task
- Execute code changes.
- Sequential or parallel implementation.

## 4. Verify
- Run tests.
- Perform UAT.
- Validate against original plan.
